Reaper Json to CSV Music Converter

How to use this converter ▼

Instructions:

  1. Upload your song_genres.json file from the \AppData\Roaming\REAPER\Scripts\ReconTracks
  2. The converter will parse the data and extract Artist, Song Name, and Genre
  3. Review the data in the table below
  4. Click "Download CSV" to save the CSV file
  5. Import the CSV file into Google Sheets
  6. Upload the data to your Song request platform like streamersonglist

ReconTracks
